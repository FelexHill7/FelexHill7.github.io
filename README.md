# FelexHill7.github.io

Personal website for GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or use a simple local server.

Example with Node:

```bash
npx serve .
```

## Deploy to GitHub Pages

Because this repository is named `FelexHill7.github.io`, GitHub serves it as a user site at:

`https://felexhill7.github.io/`

1. Commit and push these files to the `main` branch.
2. In GitHub, open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for deployment.

## Customize

- Edit text/content in `index.html`.
- Change colors and layout in `styles.css`.
- Add interactions in `script.js`.
